Import Bill of material-BOM from CSV/Excel file Odoo Apps

This Odoo apps helps to import Bill of material(BOM) with BOM lines in Odoo ERP. This Odoo apps provide feature for Bill of materials import from excel or csv file inculding its BOM lines with other fields of Bills of Material view.

If you want to import extra fields from default odoo apart from our sample files, we will add those on our import module without any extra cost. You can purchase our module and contact us on sales@browseinfo.in
e.,g If you want to add Bom Type or routing on bom, we will add without any cost.

Features

Import Bulk Bill of Material at a Time

Import bulk Bill of material and material line in onetime.

Option for BOM type

Perfect linked Bill of material and Bill of material as per data file and Option to have select type of BOM.

Import Product Option

Import products by it's Name/ Barcode or Code(Internal Reference).

Different Warning

Easy Warning when import fail due to wrong data.
